# 快速上手

完整的项目由前后端组成，由于前后端分离，需分别创建项目，便于开发调试。

# 客户端

# 环境准备

注意

请确保你的 Node.js 版本 >= 8.9， (推荐 8.11.0+)。

项目基于vue-cli3进行创建，并通过加载geelato插件的方式来启用。

  • 安装vue-cli3

  • 创建一个项目

  • 加载geelato插件 @vue/cli-plugin-geelato-ant-admin

    vue add @vue/cli-plugin-geelato-ant-admin
    1

# 服务端

# 环境准备

注意

请确保已安装jdk1.8，mysql5.7+，Redis

# 下载项目

下载项目geelato-web-quickstart，假设下载到D:\myws\geelato-web-quickstart-master。

# 创建数据库

创建数据库，建议名称为geelato。

# 修改配置

进入项目目录，打开src\main\resources目录下的application.properties，修改数据库连接信息。

# 构建项目

进入项目目录，执行bin目录下的build.bat。

# 服务端（作为代码贡献者）

创建目录D:\myws\geelato-projects\backend，下载以下项目源码到该目录。

  • geelato-core
  • geelato-utils
  • geelato-web-platform
  • geelato-web-quickstart
  • geelato-parent

在该目录下创建文件pom.xml，如下

<?xml version="1.0" encoding="UTF-8"?>
<project xmlns="http://maven.apache.org/POM/4.0.0"
         xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
         xsi:schemaLocation="http://maven.apache.org/POM/4.0.0 http://maven.apache.org/xsd/maven-4.0.0.xsd">
    <modelVersion>4.0.0</modelVersion>

    <groupId>org.geelato</groupId>
    <artifactId>geelato-project</artifactId>
    <packaging>pom</packaging>
    <version>1.0.0-SNAPSHOT</version>

    <modules>
        <module>geelato-core</module>
        <module>geelato-utils</module>
        <module>geelato-web-platform</module>
        <module>geelato-web-project</module>
        <module>geelato-web-quickstart</module>
        <module>geelato-parent</module>
    </modules>

</project>
